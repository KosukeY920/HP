ウマ娘5thライブGAZEに行ってきた

9月 19, 2023

2023年9月16日にポートメッセなごやで開催された「ウマ娘 プリティーダービー 5th EVENT ARENA TOUR GO BEYOND -GAZE-」に行ってきたぞ

会場と運営がクソ

以下クソポイント

  • 最寄り路線が1路線しか無い

    • しかも短い

  • 物販と入場待機列を一緒に並ばせるせいでカオス

    • 駅からアホほど歩いて熱すぎ死した。

  • 入場時の本人確認、荷物検査がガバガバどころかスカスカ

    • やる気のない大学生バイトくん、クソデカスーツケースすらもスルー
